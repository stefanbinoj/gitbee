## Reporting a Vulnerability

We take the security of GitBee seriously. If you have found a security vulnerability in GitBee, please report it to us as soon as possible.

### How to Report

Please do NOT report security vulnerabilities via public GitHub issues. Instead, please email us at [security@gitbee.ai](mailto:security@gitbee.ai) (or the maintainer's email).

Please include as much information as possible to help us verify and reproduce the issue, such as:

- Steps to reproduce
- Affected versions
- Potential impact

We will review your report and respond within 48 hours. We ask that you verify the issue is not public knowledge before reporting it.
